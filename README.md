**Power BI Sales Dashboard**

📊 Project Overview

This project presents a sales dashboard created using Power BI.
It includes key insights like:

- Total sales

- Number of orders

- Category performance

🔧 Tools Used

- Power BI
- Excel
- SQL

📈 Features

- KPI Cards (Total Sales, Average Sales per Order)
- Category-wise analysis
- Region-wise analysis
